
# TAKE - linie lotnicze

Projekt z przedmiotu Tworzenie Aplikacji Klasy Enterprise.

## Poradnik uruchamiania

W folderze projektu uruchomić konsolę cmd
```
docker-compose up -d --build
```
Zaczekać na stworzenie kontenera...

Uruchomić stronę [swaggera](http://localhost:8080/swagger-ui/index.html):
```
http://localhost:8080/swagger-ui/index.html
```

lub 

```
http://localhost:8080
```
